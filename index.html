What are we planning ? I think a drone.

Our main goal is to get a drone from the ground. When it is accomplished, there are possibilities to add a few features. We were thinking to add a collapsable landinggear, some lights, a camera and some other things if it's possible.

Why the hell are we doing this ? Because we needed to.

This project started out for a school course named STEM. The purpose of this project was to do self-study and learn to work in a team. It doesn't even matter if it fails or not. So we had to decide what we would make, because we both low flying, we choose to build a drone. We immediately started to dream, but we had to stay realistic. We don't want to fail and how bigger we see it, how more it was doomed to fail. First, we would do everything ourselves, but it was slightly out of league. After that, we decided to first get a drone in the air and only then we can start to add some features.
